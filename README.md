# CPE-and-Other-Practice
CPE 程式設計檢定練習與題解與其他練習題練習

### 1. 3n+1
* 7/10完成
* **程式運作說明** :
*   **輸入** : 每行輸入包含兩個正整數 `i` 和 `j` ， 程式會持續讀取直到檔案結束（EOF），所以我用 while 來輸入。
*   **輸出** : 對應每組輸入，依序輸出原始的 `i` 和 `j` 以及該區間內的最大循環長度 ， 數值間以空格分隔並換行。

### 2. Odd_Sum
* 7/10完成
* **程式運作說明** :
*   **輸入** : 輸入的第一列有一個整數 `t` (1<=t<=100)， 代表以下有多少組測試資料 。 每組測試資料為兩列，包含兩個數 `a` 和 `b`。(0<=a<=b<=100)
*   **輸出** : 每組測試資料輸出一列，內容為 `a`及 `b 間所有奇數的和。
